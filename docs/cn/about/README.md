## 这个网站是什么？
静竹林心灵网站，是一个在公元两千年初，由吾师静竹林所创立的论坛。
网站成立的目的，是为了让曾有心灵、修行困扰的人，能找到一个可以交流的地方。

从最初的八大电视《神出鬼没》节目论坛、如庸论坛，到后续由网友协助迁移至雷傲讨论区与 LeoBBS；
之后再由现协会成员搬迁到 Discuz 系统。
主机也从大陆迁回台湾，网站随着时代经历了许多变迁。

如今，论坛形式逐渐被各种社群媒体取代。
因此我们转向静态网站形式，希望通过 GitHub 托管，尽可能长期保存这些年来的记录。

在立场上，本站管理者期待两岸和平。
管理者中有祖籍湖南与福建、在台湾成长的成员。
我们不希望放大对立，而是希望把真实经历整理为可交流、可反思的资料。

### 是诈骗网站吗？
首先，我们并没有一群人聚在一起，
像教派一样要求大家信仰某个东西，或要求大家捐款。

我们只是一群曾经有心灵困扰，
或对佛法、道法、心灵成长有兴趣的人，聚在一起互相交流。
绝大多数人都是在网络上找到这个网站，从而认识静竹林，并解决灵性上的各种困扰。
网友们也从网络关系变成真实朋友，持续在生活中交流。

而论坛内的文章与对话，成为大家走过各自历程后留下的记录。
这些内容，或许会对未来看到网站的人有所帮助，所以我们坚持把它们保留下来。

### 静竹林是谁？
静竹林，是我们的老师，一位住在台湾台北的装修木工师傅。
他凡事以理性分析，在生活中处处以观察、验证、归纳的方式，
把人生中的各种问题，用近似电脑科学的方法来处理，
留下许多有智慧的洞见，并且身体力行、说到做到。

据我们与他多年的交流，他是在修行上有深厚造诣的人；
在现代社会较少被公开谈及的说法中，他也是一位能力很高的通灵者。

但他始终只是个普通人。
无论曾有灵界要他通灵办事，或电视节目想把他塑造成明星，
他都只愿平凡生活，当一位非常务实的木工职人。
他总是劝人不要迷信，先从医学、科学角度思考并解决问题。

在如今的生活里，他低调从事装修工作，随缘结识大众、随缘帮忙。
这里没有高高在上的老师，也没有偶像崇拜，
只有一群人互相打气、互相帮忙。
有时彼此调侃，跌倒时再轻轻踢你一脚，
提醒你记取教训，重新站起来，一起成长。

### 这群朋友都是什么人？
这群朋友来自各行各业。
有人加入时是高中生、大学职员、小学英文老师、道上兄弟，或在大陆工作的台干等。

如今，当年的高中生已成为系统工程师、大学讲师，也有人如愿考上药师；
有人成家立业、已有儿女，也有人在人生途中早已离开人世。

最初加入的目的，或多或少都是为了解决心灵困扰，
例如灵扰、卡阴、卡神等问题，
最后大家都找到各自方式，平凡地在社会中继续生活。

而在这个网站上，多数网友都曾有类似困扰经验。
但这类问题横跨身、心、灵三个领域，
灵学部分甚至是现代科学尚未深入的范畴。

若身心疾病又叠加灵学问题，
整个折磨过程往往非常无助。
到处寻求解答，害怕被骗；
四处碰壁，又担心情况更严重。
这种磨难过程，确实很难被一般人理解。

大家曾共同走过这些阶段，彼此交流、彼此帮助，
也在过程中留下文字记录。
几年后回头看看过去的自己，
会不会又气又好笑呢？

### 这些文章是真的吗？
在现代社会，愿意阅读、接收讯息，
乃至相信并采取行动，是非常困难的事。
其中有两个关键：第一是讯息真实性，第二是讯息可信度。

真实性：这些文章是从封存论坛内容复制而来。

现代社会的确有很多假消息、假新闻，
甚至在 AI 时代还有大量伪造文章。
当我们发布这个网站时，
许多科技巨头也在同步搜集训练资料，
网站上的文章都可能成为 AI 训练素材。

而我们的文章，或许有些排版错乱，
但如果封存论坛仍可运行，就能看见这些真实的人们，
跨越二十多年时光所留下的记录。

可信度：本站文章由一群人共同讨论、共同书写而成。

灵学本来就是现代科学尚无法完整解释的领域。
举例来说，我们能够理解一维、二维乃至三维空间，
但超过三维空间，我们就很难理解。

进一步说，现代 AI 科技依赖深度学习，
其中会把三维、四维甚至更高维的数据进行多维暂存，
再由不同类神经网络算法对资料进行分类或预测。

这是现代科学中的一个例子。
我们能理解物质存在，但对三维以上空间难以直观理解；
难以理解不代表不能探索，只能尽量用更抽象、数学化方式描述。
同样地，要认知与理解灵魂存在，
也必须保留一些想像空间，去比喻、去思考。

正因如此，吾师也常说，他不会轻易否定别人对灵界的体验与见解。
毕竟我们讨论的对象，已经超出现代科学既有范畴；
又因每个人经验、缘分、智慧不同，观点自然各异。
因此，多元声音反而是一种丰富。
在丰富文章里，自然能找到基本共识，
并形成你个人对灵学的见解。
